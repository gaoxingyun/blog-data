## angular-cli使用

#### 命令

###### 新建项目
```
# 新建一个项目
ng new $PROJECT_NAME
# 新建一个项目，使用自定义样式格式scss
ng new $PROJECT_NAME --style scss
# 新建一个项目，使用自定义样式格式sass
ng new $PROJECT_NAME --style sass
```
